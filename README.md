# Chaincast Contracts

This repository contains all the smart contracts needed to run the Chaincast service.

## Hardhat Cheatsheet

### Start Local Network

```
npx hardhat node
```

### Deploy on Local

```
npx hardhat run scripts/deploy.js --network localhost
```
